## ✨ Features

[PaperBanana](https://paper-banana.net/) is an intelligent platform that transforms complex research concepts into publication-ready academic diagrams. Whether you're drafting a paper for NeurIPS, ICML, or Nature, our AI understands your scientific context.

### 📊 Text-to-Diagram
Convert text descriptions into structured architecture diagrams or flowcharts instantly.

### ✏️ Sketch-to-Vector
Upload rough hand-drawn sketches and get polished vector-quality illustrations.

### 📈 Statistical Plots
Generate accurate charts from raw data with professional color palettes and typography.

### 🎨 Journal-Standard Aesthetics
Outputs optimized for readability and the aesthetic standards of top-tier journals.

### Key Capabilities

| Feature | Description |
|---------|-------------|
| 🤖 Multi-Agent System | 5 specialized agents: Retriever, Planner, Stylist, Visualizer, Critic |

| ⚡ Fast Generation | 5 hours in PowerPoint → 5 seconds with PaperBanana |

## 🚀 Quick Start

### Online Access (Recommended)

The easiest way to experience PaperBanana:

1. Visit [paper-banana.net](https://paper-banana.net/)
2. Enter your text description or upload a sketch
3. Select your style (Clean Tech, Classic Journal, Modern AI)
4. Click "Generate" and download your publication-ready figure!

> No installation required • Free tier available • Vector output supported

```

│  Input Layer  │  Text / Sketch / Data                   │
```
